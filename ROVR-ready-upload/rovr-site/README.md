
# ROVR (Ready-to-Upload)

This folder is a plug-and-play Next.js 14 + Tailwind project for the ROVR landing page.

## Quick Deploy (No Git Required)
1. Unzip this folder.
2. Go to https://vercel.com/new → Import Project → **Upload** (top right).
3. Drag the unzipped folder in and click **Deploy**.

That's it. Vercel will install dependencies and build the site.

## Local Preview (optional)
If you want to run it locally:
```bash
npm install
npm run dev
# open http://localhost:3000
```
